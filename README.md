# Cara Menjalankan Proyek Laravel

Ikuti langkah-langkah berikut untuk menyiapkan dan menjalankan proyek Laravel:

1. clonw repository:

    ```bash
    git clone <my-cool-project>
    ```

2. ```bash
   composer install
   ```

3. ```bash
   cp .env.example .env

   ```

4. ```bash
   php artisan key:generate
   ```

5. ```bash
   php artisan migrate:refresh --seed
   ```

6. ```bash'
   php artisan serve

   ```
